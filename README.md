# Name

    - API to upload CSV file to database

# Description

    - a rest api to  upload  a csv file into database , get save file and download from database as csv

## Stack

    - Node / Express
    - Maria DB
    - sequelize
    - Multer

## Installation

```bash
    npm install
```

## RUN

### Development

```bash
 npm start:dev
```

### Production

```bash
npm start
```
